# Laravel Frontend Guidelines

## Stack

* Laravel
* Vue 3
* Composition API
* Pinia
* Vue Router
* TailwindCSS
* Vite

## Architecture

* Use modular architecture
* Separate public and admin areas
* Use reusable components
* Keep components small
* Use services for API requests
* Use composables for reusable frontend logic
* Use stores only for global state
* Keep pages clean and organized

## Patterns

* Use script setup syntax
* Use TypeScript whenever possible
* Prefer composition over large components
* Use layout-based architecture
* Separate concerns properly

## Folder Structure

* pages
* components
* layouts
* stores
* services
* composables
* router

## UI

* Modern UI
* Responsive design
* Coca-Cola campaign visual identity
* Clean spacing
* Mobile-first approach

## Public Area
